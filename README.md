# ANIMA Web Prototipi

ANIMA, oyun hissi taşıyan bir öğrenme uygulaması deneyimi için hazırlanmış bir tasarım prototipidir. Bu proje, üretime hazır backend özelliklerinden çok arayüz tasarımına, görsel yöne ve kullanıcı akışına odaklanır.

Prototip React ve Vite ile geliştirilmiştir. Profil seçimi, öğrenme kategorileri, dersler, alıştırma ekranları, rozetler ve uygulama içi gezinme gibi örnek ekranları içerir.

## Yerelde Çalıştırma

```bash
npm install
npm run dev
```

## Build Alma

```bash
npm run build
```

## Not

Bu repository bir UI/UX tasarım prototipidir. Veriler, profiller, puanlar ve öğrenme içerikleri ürün fikrini göstermek için kullanılan örnek değerlerdir.
